## What Makes The Frame TV Different to Install

### 1. The No Gap Wall Mount (Slim Fit Mount)

The Frame TV is designed to sit **flush against the wall with zero gap** — like an actual framed artwork. This requires Samsung's official No Gap Wall Mount (sometimes called the Slim Fit Mount).

Standard VESA mounts work mechanically, but they create a gap between the TV and the wall that breaks the picture-frame illusion. If the flush look matters to you — and it usually does for Frame TV buyers — you need the Samsung-specific mount.

**Important:** We install customer-supplied mounts. We do not sell brackets directly. Make sure you order the right generation of the No Gap Mount for your Frame TV model year (Samsung updates these annually).

### 2. The One Connect Box

Unlike standard TVs where all inputs are on the back, The Frame TV routes everything through an external **One Connect Box** connected by a single transparent cable. This box holds your HDMI ports, USB, and all other connections.

The box needs to go somewhere — and that somewhere matters:
- Behind the TV itself (some mounts allow this)
- Inside a nearby media cabinet
- Behind furniture below the TV
- In a wall cavity if you are doing a full in-wall routing job

We plan the One Connect Box placement with you during the appointment. Getting this right is the difference between a clean install and a box sitting visibly on a shelf.

### 3. The Transparent Cable

Samsung's transparent cable running from the TV to the One Connect Box is intentionally low-profile, but it is not invisible. Options for managing it:

- **Route in-wall**: completely hidden, cleanest result, works on drywall
- **Surface raceway**: slim channel on the wall surface, can be painted to match
- **Leave as-is**: the transparent cable is nearly invisible close to the wall, some homeowners find it acceptable

[Cable concealment](/cable-concealment-nashville) is a $60 add-on and handles the transparent cable as part of the package.

## Popular Install Scenarios for The Frame TV

### Above a Fireplace

The Frame TV above a fireplace is one of our most common Nashville requests — it looks exactly like a painting hanging above the mantel. We verify heat clearance, handle the One Connect Box routing, and can manage the cable path along the fireplace wall.

Read more: [TV mounting over fireplace in Nashville](/tv-mounting-over-fireplace-nashville)

### Gallery Wall Layout

The Frame TV blends into a gallery wall with framed photos and artwork. We handle the precision leveling required so the TV sits perfectly aligned with surrounding pieces.

### Bedroom Feature Wall

In a master bedroom, The Frame displaying art overnight is a popular design choice. We mount it at the right height for in-bed viewing and route the cables completely in-wall.

## Art Mode Setup

After mounting, we connect the One Connect Box, test all inputs, and walk you through the Art Mode setup before leaving. Art Mode allows the TV to display artwork or photos continuously when the TV is off — we make sure it is working before we go.
